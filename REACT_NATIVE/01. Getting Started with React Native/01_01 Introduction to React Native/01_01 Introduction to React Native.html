01_01 Introduction to React Native - Q&A Set 1

1. What is React Native?

React Native is an open-source framework developed by Facebook that allows developers to build mobile applications using JavaScript and React. Unlike traditional hybrid apps, React Native compiles to native app components, which enhances performance and user experience.

// React Native combines the best parts of native development with React
// You write components in JavaScript, but they render as native views on iOS and Android

2. How do you create a new React Native project?

You can create a new project using npx and the React Native CLI:

npx react-native init MyNewApp

This will scaffold a new React Native project with default configurations.

3. What are the basic building blocks in React Native?

Some of the core components in React Native include:

  • View: The fundamental building block for layout
  • Text: For displaying text
  • Image: For displaying images
  • ScrollView: For scrollable content
  • TextInput: For user input
import React from 'react';
import { View, Text, Image, TextInput, ScrollView } from 'react-native';

const App = () => (
  <ScrollView>
    <View>
      <Text>Welcome to React Native!</Text>
      <Image source={{ uri: 'https://example.com/image.png' }} style={{ width: 100, height: 100 }} />
      <TextInput placeholder="Enter text" style={{ borderWidth: 1 }} />
    </View>
  </ScrollView>
);

export default App;

4. What is the difference between React and React Native?

React is used for building web applications, while React Native is used for building mobile apps. React uses HTML and CSS, whereas React Native uses components that render to native views and uses StyleSheet for styling.

// React (web)
<div style={{ color: 'red' }}>Hello Web</div>

// React Native (mobile)
<Text style={{ color: 'red' }}>Hello Mobile</Text>

5. How do you run a React Native app on an emulator?

Once you have created your app with the CLI, you can run:

npx react-native run-android
npx react-native run-ios

This will launch the Android emulator or iOS simulator (requires Xcode on macOS).

JSX is a syntax extension that allows writing HTML-like code in JavaScript. In React Native, JSX is used to describe the UI.

8. How do you handle user input in React Native?

You can use TextInput and manage its value with state.

import React, { useState } from 'react';
import { View, TextInput, Text } from 'react-native';

const App = () => {
  const [value, setValue] = useState('');

  return (
    <View style={{ padding: 20 }}>
      <TextInput
        style={{ borderWidth: 1, padding: 10 }}
        placeholder="Type here"
        value={value}
        onChangeText={setValue}
      />
      <Text>You typed: {value}</Text>
    </View>
  );
};

export default App;

14. How do you use ScrollView in React Native?

ScrollView is a component that allows scrolling content vertically or horizontally. It should be used for smaller content that fits within memory.

import React from 'react';
import { ScrollView, Text } from 'react-native';

const App = () => (
  <ScrollView>
    {Array.from({ length: 20 }).map((_, i) => (
      <Text key={i}>Item {i + 1}</Text>
    ))}
  </ScrollView>
);

15. What is FlatList and when should you use it?

FlatList is optimized for long lists of scrollable data, rendering items lazily for better performance.

import React from 'react';
import { FlatList, Text, View } from 'react-native';

const DATA = Array.from({ length: 100 }, (_, i) => ({ id: i.toString(), title: `Item ${i}` }));

const App = () => (
  <FlatList
    data={DATA}
    keyExtractor={item => item.id}
    renderItem={({ item }) => (
      <View>
        <Text>{item.title}</Text>
      </View>
    )}
  />
);

18. How do you use useState in React Native?

useState is a hook to manage local component state.

19. How do you use useEffect in React Native?

useEffect lets you run side effects in functional components.

import React, { useState, useEffect } from 'react';
import { View, Text } from 'react-native';

const App = () => {
  const [time, setTime] = useState(new Date().toLocaleTimeString());

  useEffect(() => {
    const interval = setInterval(() => {
      setTime(new Date().toLocaleTimeString());
    }, 1000);
    return () => clearInterval(interval); // Cleanup on unmount
  }, []);

  return <Text>Time: {time}</Text>;
};

24. How to fetch data in React Native?

Use fetch() inside useEffect and store the result in state.

import React, { useEffect, useState } from 'react';
import { View, Text } from 'react-native';

const App = () => {
  const [data, setData] = useState(null);

  useEffect(() => {
    fetch('https://jsonplaceholder.typicode.com/posts/1')
      .then(res => res.json())
      .then(json => setData(json));
  }, []);

  return <View><Text>{data?.title}</Text></View>;
};

26. What is the role of AppRegistry?

AppRegistry is the entry point to run all React Native apps. It tells React Native which component is the root of the application.

AppRegistry.registerComponent(appName, () => App);

Use FlatList instead of ScrollView, avoid inline functions, use memoization (React.memo, useCallback), and reduce unnecessary re-renders.

  return <Text>{value}</Text>;
});

const memoizedCallback = useCallback(() => {
  // some action
}, []);
