25 JSX Syntax Questions with Answers and Detailed Code

1. What is JSX in React?

Answer: JSX stands for JavaScript XML. It is a syntax extension that allows you to write HTML tags inside JavaScript code, which React uses to describe the UI.

const element = <h1>Hello, world!</h1>;

2. Can JSX return multiple elements?

Answer: JSX must return a single parent element. To return multiple elements, you can wrap them in a parent element or a React Fragment.

const App = () => {
  return (
    <div>
      <h1>Title</h1>
      <p>Description</p>
    </div>
  );
};

// Or using Fragment
const App = () => {
  return (
    <>
      <h1>Title</h1>
      <p>Description</p>
    </>
  );
};

3. How do you embed JavaScript expressions in JSX?

Answer: JavaScript expressions can be embedded in JSX using curly braces { }.

const name = 'Manuela';
const element = <h1>Hello, {name}!</h1>;

4. Can you use if-else statements in JSX?

Answer: You can't use if-else directly inside JSX, but you can use ternary operators or conditional expressions.

const isLoggedIn = true;
const message = (
  <h1>{isLoggedIn ? 'Welcome back!' : 'Please sign in.'}</h1>
);

5. What is the use of className in JSX?

Answer: In JSX, use className instead of class to define CSS classes because class is a reserved word in JavaScript.

const element = <div className="container">Content</div>;

6. How do you add inline styles in JSX?

Answer: Inline styles in JSX are specified as an object with camelCase property names.

const element = <div style={{ color: 'blue', fontSize: '20px' }}>Styled Text</div>;

7. How does JSX handle events?

Answer: JSX uses camelCase for event names and passes a function as the event handler.

const handleClick = () => {
  alert('Button clicked');
};

const element = <button onClick={handleClick}>Click Me</button>;

8. Can you use comments in JSX?

Answer: Yes, comments can be written inside curly braces using JavaScript comment syntax.

const element = (
  <div>
    {/* This is a comment */}
    <p>Hello JSX</p>
  </div>
);

9. How do you render a list in JSX?

Answer: Use the map() method to render a list of elements in JSX.

const items = ['Apple', 'Banana', 'Cherry'];
const element = (
  <ul>
    {items.map(item => <li key={item}>{item}</li>)}
  </ul>
);

10. What are fragments in JSX?

Answer: Fragments let you group multiple elements without adding an extra node to the DOM.

const element = (
  <>
    <h1>Title</h1>
    <p>Subtitle</p>
  </>
);

11. How do you handle boolean attributes in JSX?

Answer: In JSX, boolean attributes like disabled can be written with just the attribute name set to true.

const element = <button disabled>Disabled Button</button>;

12. Can you nest elements in JSX?

Answer: Yes, JSX allows you to nest elements just like HTML.

const element = (
  <div>
    <h1>Hello</h1>
    <p>Welcome to JSX</p>
  </div>
);

13. How does JSX prevent injection attacks?

Answer: JSX escapes any values embedded in it before rendering, which helps prevent XSS attacks.

const title = '<script>alert("Hacked")</script>';
const element = <h1>{title}</h1>; // Rendered safely

14. What is the role of keys in JSX lists?

Answer: Keys help React identify which items have changed, been added, or removed. They should be unique among siblings.

const items = ['a', 'b', 'c'];
const list = items.map((item, index) => <li key={index}>{item}</li>);

15. How do you use expressions in attribute values in JSX?

Answer: You can use curly braces {} to embed JavaScript expressions in attribute values.

const imageUrl = 'https://example.com/logo.png';
const element = <img src={imageUrl} alt="Logo" />;

16. How do you pass functions as props using JSX?

Answer: You can pass functions to components as props by using curly braces.

const Button = (props) => (
  <button onClick={props.onClick}>Click</button>
);

const handleClick = () => alert('Clicked!');

const App = () => <Button onClick={handleClick} />;

17. Can JSX be used without React?

Answer: JSX is designed to work with React, but it can be compiled and used with other frameworks by transforming it to regular JavaScript.

// Compiled JSX
React.createElement('h1', null, 'Hello');

18. How do you handle conditional rendering with logical AND in JSX?

Answer: You can use the logical AND (&&) operator to conditionally render elements.

const isLoggedIn = true;
const element = (
  <div>
    {isLoggedIn && <h1>Welcome!</h1>}
  </div>
);

19. What is JSX transpilation?

Answer: JSX code is transpiled to regular JavaScript using tools like Babel before it can run in the browser.

// JSX
const element = <h1>Hello</h1>;

// Transpiled
const element = React.createElement('h1', null, 'Hello');

20. How do you handle default props in JSX components?

Answer: You can assign default props to a component using the defaultProps property.

const Greeting = (props) => (
  <h1>Hello, {props.name}!</h1>
);

Greeting.defaultProps = {
  name: 'Guest'
};

21. How can you use a ternary operator in JSX?

Answer: Use the ternary operator to render content conditionally.

const isLoggedIn = true;
const element = (
  <div>
    {isLoggedIn ? <h1>Welcome</h1> : <h1>Please sign in</h1>}
  </div>
);

22. What are fragments in JSX and how are they used?

Answer: Fragments let you group a list of children without adding extra nodes to the DOM.

const element = (
  <React.Fragment>
    <h1>Title</h1>
    <p>Description</p>
  </React.Fragment>
);

// Short syntax
const element2 = (
  <>
    <h1>Title</h1>
    <p>Description</p>
  </>
);

23. How can you comment inside JSX?

Answer: Comments in JSX use curly braces with // or /* */ syntax.

const element = (
  <div>
    {/* This is a comment */}
    <h1>Hello</h1>
  </div>
);

24. How do you dynamically apply classes in JSX?

Answer: Use JavaScript expressions to conditionally set class names.

const isActive = true;
const element = <div className={isActive ? 'active' : 'inactive'}>Status</div>;

25. Can you inline styles in JSX?

Answer: Yes, you can pass a JavaScript object to the style attribute in JSX.

const divStyle = {
  color: 'blue',
  backgroundColor: 'lightgray'
};

const element = <div style={divStyle}>Styled Text</div>;
